# Questions
É uma aplicação de perguntas e respostas.
Nela é possível: 
  - navegar pelas perguntas
  - criar uma nova
  - consultar as respostas feitas pela comunidade 
  - responder a uma pergunta
  - reagir a uma pergunta ou resposta
  - campo de busca
  
Tecnologias:
  - React.js
  - Golang
  - MongoDB
  - Docker
 
 Pré-requisitos:
  Possuir o docker instalado. 
 
 Start
 ```
 docker-compose up --build
 ```
 Run
 ```
 docker-compose up
 ```
 Stop
 ```
 docker-compose down
 ```
